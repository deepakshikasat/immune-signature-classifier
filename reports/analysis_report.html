Immune Gene Signature Classifier

Bench to Bytes portfolio analysis report. Public data sources are named explicitly.

Hypothesis

A credible IFN classifier should validate selected genes against public canonical interferon-response signatures.

Data

Synthetic IFN expression matrix plus public canonical IFN alpha/gamma response gene symbols used as an external biological validation set.

Process

  1. Added a public IFN signature reference table.
  2. Merged ranked genes with public IFN alpha/gamma membership.
  3. Replaced the figure with a top-25 overlap summary.
  4. Reframed the repo as partially public-validated, not fully public RNA-seq.

Outputs

  • outputs/gene_signature_ranking.csv

Interpretation

The project is more honest and biologically grounded. The next real upgrade is still a true GEO/DEE2 IFN-treated RNA-seq dataset.
