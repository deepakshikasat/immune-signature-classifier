Immune Gene Signature Classifier

Bench to Bytes portfolio analysis report. Uses only public or synthetic data.

Hypothesis

IFN-treated samples should be distinguishable by a compact gene signature enriched for canonical interferon-stimulated genes.

Data

Synthetic IFN-stimulation expression matrix using canonical public ISG symbols such as MX1, IFIT1, ISG15, OAS1, STAT1, and IRF7.

Process

  1. Generated control and IFN-treated expression profiles.
  2. Computed treated-control log2 fold-change for each gene.
  3. Annotated known ISGs and ranked features by effect magnitude.
  4. Plotted the top signature genes with clear labels.

Outputs

  • data/expression_log2.csv
  • data/sample_metadata.csv
  • outputs/gene_signature_ranking.csv
  • figures/top_signature_genes.svg

Interpretation

The ranking recovers several canonical ISGs while also showing non-canonical synthetic features, matching the real-world task of separating known biology from model-suggested candidates.
